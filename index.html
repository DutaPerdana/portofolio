About Me

Halo, saya Duta Perdana, seorang mahasiswa Teknik Komputer di Politeknik Negeri Medan. Saya sangat tertarik dengan dunia programming, terutama di bidang web development. Saya sangat bersemangat untuk terus belajar dan berkembang di dunia teknologi, terutama dalam bidang web development. Setiap tantangan adalah kesempatan bagi saya untuk meningkatkan keterampilan dan menambah pengetahuan baru.

Project 5

Game ini memungkinkan pemain untuk bermain melawan komputer dengan aturan dasar suwit.
